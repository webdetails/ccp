Be part of the Community

We understand that for a Community to be successful we have to foster collaboration and ensure success. We want our users to get help and contribute back with that knowledge. And here are some ways how you can do it, and by the way - try the live chat.

Looking for help?

First stop is always the Pentaho Documentation. But there are other ways to get engaged. Here are some possibilities

Forums

The Pentaho forums are an excellent resource not only to get help from experts but also to pass on your expertise to less experienced users. Here are the most popular ones. Did you help someone today?

  • BI Platform, which enables the delivery of a unified, end to end solution from data integration to visualization and consumption of data.
  • Data Integration [Kettle], core data integration (ETL) engine, and GUI applications to allow users to define data integration logic
  • Analysis [Mondrian], open source, Java, OLAP (online analytical processing) server
  • Reporting, a visual, banded report writer for creating printable pixel perfect reports
  • Dashboard [Ctools], the set of tools that allows to build highly customized dashboards
  • Data Mining [Weka], consisting of machine learning algorithms for a broad set of data mining tasks
  • ... and many more
